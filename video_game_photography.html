In Game Photography

Tools and How To Use the Tools

The five images above are photos that I took using Otis_inf photomode tool and mods for this game that can be found in Nexus Mods

Otis_inf tool is basically a photo-mode injection tool which runs while the game is running in order to work together.

Besides the injection tool, I also have to use an app called Reshade where I can manipulate in-game assets such as image sharpening, light settings, field of view, lens aperture, and long exposure units.

Tutorial

The video attached is made by a YouTuber called Shadria, who demonstrates an overall idea as to how to use the tools I have mentioned above and also use other external tools in order to get the desired shot.
